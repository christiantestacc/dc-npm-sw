# What is this

This is an npm package based off of DesignCourse's youtube tutorial.

# Installation

`npm i shadowizard`

Then...

```
import { shadowizard } from 'shadowizard'

shadowizard({
  shadowType: 'soft',
  padding: false,
})
```

## Options

* *shadowType* _hard || soft_ (Defaults to soft)
* *padding* _boolean_ (Defaults to false)
